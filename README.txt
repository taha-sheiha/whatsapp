📜 دليل تشغيل بوت واتساب النيورا (Render Deployment) 🚀
====================================================
آخر تحديث: مارس 2026 - الإصدار المستقر v3

1️⃣ إعدادات موقع Render:
-----------------------
عشان السيرفر يشتغل صح، تأكد من الإعدادات دي في لوحة التحكم:
- الـ Root Directory: (سيبه فاضي Empty)
- الـ Build Command: `cd bot && npm install`
- الـ Start Command: `cd bot && npm start`
- الـ Node Version: (على الأقل v18)

2️⃣ المتغيرات البيئية (Environment Variables):
--------------------------------------------
في صفحة Environment بموقع Render، ضيف المتغيرين دول:
- WORKER_SESSION_URL: (الرابط الخاص بـ Cloudflare Worker للرسايل)
- AI_API_URL: (الرابط الخاص بردود الذكاء الاصطناعي)

3️⃣ التشغيل والربط (Linking Method):
------------------------------------
- بعد ما السيرفر يكون Live، افتح الرابط بتاعه في المتصفح.
- عندك طريقتين للربط:
  أ) مسح الباركود (QR Code) - تقليدية.
  ب) طلب كود الربط (Pairing Code) - "الأفضل والأسرع":
     1. اكتب رقمك الدولي (مثال: 2010... أو 9665...)
     2. انسخ الكود اللي هيظهر لك.
     3. افتح واتساب من موبايلك -> الأجهزة المرتبطة -> ربط جهاز -> الربط برقم هاتف بدلاً من ذلك.
     4. دخل الكود ومبروك!

4️⃣ حل المشكلات الشائعة (Troubleshooting):
------------------------------------------
لو واجهت "خطأ في الاتصال" أو "401 Unauthorized":
1. افتح صفحة التحكم الخاصة بالبوت مجدداً.
2. هتلاقي زرار أحمر اسمه "مسح الجلسة تماماً (Reset)".
3. دوس عليه مرة واحدة، السيرفر هيعمل تنظيف تلقائي للداتا القديمة.
4. اطلب كود ربط جديد وهيشتغل فوراً.

5️⃣ منع السيرفر من النوم (Keep-Alive):
-------------------------------------
استخدم Cron-job.org لضرب رابط السيرفر بتاعك /ping كل 10 دقائق لضمان الاستجابة السريعة 24/7.

6️⃣ إعدادات المطور (API URLs Locations):
------------------------------------
لو حبيت تغير الروابط يدوياً من داخل الكود، دي الأماكن بالظبط:

أ) رابط الذكاء الاصطناعي (Gemini Chat):
   - الملف: `bot/listener.js`
   - السطر: رقم 7 (المتغير `AI_API_URL`).

ب) رابط حفظ الجلسات (D1 Session):
   - الملف الأول: `bot/session_remote.js` في السطر رقم 5.
   - الملف الثاني: `bot/server.js` في السطر رقم 21 (عشان زرار الـ Reset).
